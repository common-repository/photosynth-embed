Provides easy shortcode access to inserting Photosynth panormamas and synths

== Description ==

This is a widget that allows you to quickly and easily create a photosynths within posts and pages using a simple shortcode ([synth]). The advantage of having a shortcode is that it stops problems when previeing the post/page while editing.

To be written. This product is in an early beta test!

== Installation ==

Installation is easy:

* Download and unzip the plugin.
* Copy the folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* Include a widget in a post or page by adding the appropriate shortcode to your page or post.
